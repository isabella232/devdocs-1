scipy.integrate.quad_explain¶

scipy.integrate.quad_explain(output=<_io.TextIOWrapper name='<stdout>' mode='w' encoding='utf-8'>)[source]¶

Print extra information about integrate.quad() parameters and returns.

Parameters
outputinstance with “write” method, optional

Information about quad is passed to output.write(). Default is sys.stdout.

Returns
None

Examples

We can show detailed information of the integrate.quad function in stdout:

>>> from scipy.integrate import quad_explain
>>> quad_explain()
